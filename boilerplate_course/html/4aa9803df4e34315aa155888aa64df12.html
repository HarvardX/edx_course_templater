This next problem has a lot more features:

  • It does rich text instead of plain text.
  • It saves learner data so they can see their work elsewhere in the course. Use the data-saveslot attribute to set the default file name.
  • It can pull saved work from a previous file. Use the data-from-slot attribute to set that.
  • Learners can make multiple files, save them, rename them, and delete them.
  • It can export files for the learners to save, in HTML format.

Setup is slightly more complex, but not by a ton.

  • It needs HX-JS on the page in order to work.
  • It also needs the Learner Backpack to be in place, which it is in this course.
  • You need both an HTML component with the editor, and the advanced problem beneath it for grading. You can't put the editor inside the problem, for reasons I do not yet understand.
