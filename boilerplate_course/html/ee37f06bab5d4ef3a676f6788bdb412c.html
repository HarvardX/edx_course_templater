Make Your Own Accessible Ordering Problem

This is time-consuming and tricky the first time, but it gets easier as you make more of them. If you've made a Matching problem, it will be very familiar, because the two are very similar.

  1. Come up with the items.
  2. Find images or write text (full HTML ok) for the cards. If you're using images, drop them in Files & Uploads. If you're using text, consider front-loading it so that students hear the most important or identifiable parts of them first when using a screen reader.
  3. Download this zip file and open it. You will only need some of what's inside.
  4. Rename the Ordering_Elements.js file and open it with a text editor. Fill in the 'type' property with either 'text' or 'image', and fill in the according property with the image's filename or with your text. The text field is used as alt text for images.
  5. Rename the Ordering.html file. Open it with a text editor and change where it says <script src="Ordering_Elements.js"></script> to match your new Elements file.
  6. Upload your two new js and html files to Files & Uploads.
  7. Duplicate the Ordering problem above. Rename it and move it to where you want it in the course.
  8. Edit that problem:
    1. Change where it says html_file="/static/Ordering.html" near the bottom, replacing Ordering.html with your own filename.
    2. Change where it says right_answer.append( ['ACFBE'] ) in the middle-top to be your answer. Pay close attention to the formatting. You can also add more lines like this one to have multiple right answers.
  9. Test it!
