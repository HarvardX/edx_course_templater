How-To:

Copy the code from this HTML component and place it underneath a video in your course. You can set the time in seconds for each link, or in mm:ss format. You can customize the links, the text that appears, even the image (which is stored in Files & Uploads). For more detail on settings, check out the "VidLinkOptions" settings on HX-JS, down at the bottom.

The list of links below the video is generated automatically, for accessibility purposes.
