This page has a bunch of sample text and image tricks. All of them can be combined with each other. Don't forget to include the HX-JS tags when you use them, but you only need to include them once per page.

<script src="/static/hx.js" type="text/javascript"></script>
<link rel="stylesheet" type="text/css" href="/static/hx.css" />

You probably noticed the table-of-contents hanging on the right there. That's sometimes useful on long pages like this one, and you can use <script>var hxLocalOptions = {makeTOC: true};</script> at the top of your page to make that appear if you want.
