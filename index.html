This page builds edX course templates. Put in your selections, click the "Create downloadable file" button, and you'll be able to download a tar file suitable for uploading to edX. Remember that uploading this file will overwrite anything that currently exists in your course, so use this early in your process.

To run this with a blank course instead of preexisting boilerplate, leave the Boilerplate source file line blank.

Created by Colin Fredericks for HarvardX, March 2019. Source code available on GitHub.
